# Sonia Dhiman — Portfolio

Personal portfolio website for **Sonia Dhiman**, Digital Marketing Specialist & Web Designer based in Una, Himachal Pradesh, India.

## Live Site

[soniadhiman-dev.github.io](https://soniadhiman-dev.github.io/soniadhiman-dev)

## About

A modern single-page portfolio built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies beyond Google Fonts.

**Sections:**
- Hero with animated status badge and profile card
- About with capability cards
- Skills with animated progress bars (4 categories)
- Projects with filter tabs (Web / SEO / Marketing / Branding)
- Professional Experience timeline
- Services with pricing
- Contact form
- Hire Me platform links
- Footer

## Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox) |
| Interactivity | Vanilla JavaScript (IntersectionObserver) |
| Fonts | Inter + JetBrains Mono via Google Fonts |
| Hosting | GitHub Pages |

## Features

- Fully responsive (mobile, tablet, desktop)
- Animated skill bars triggered on scroll
- Project filter system
- Smooth scroll + sticky navbar with active section highlight
- Dark theme with purple/violet accent
- Mobile hamburger menu
- Contact form with feedback state

## Local Development

No build step required — just open `index.html` in a browser.

```bash
# Clone
git clone https://github.com/soniadhiman-dev/soniadhiman-dev.git

# Open
start index.html
```

## Deploy

This site is deployed via **GitHub Pages**. Any push to `main` updates the live site automatically (once Pages is enabled in repo Settings → Pages → Source: Deploy from branch `main` / `/ (root)`).
